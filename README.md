# Shop Page – React E-Commerce UI

A modern and responsive **Shop Page** built using **React.js**, featuring dynamic product listings, filters, and a clean UI.  
This project demonstrates essential front-end e-commerce functionality using a modular component structure and public API integration.

---

## Features

- **Dynamic Product Fetching** – Products are loaded from [FakeStoreAPI](https://fakestoreapi.com/).
- **Filter Sidebar with Accordions** – Includes categories such as:
  - Ideal For  
  - Occasion  
  - Fabric  
  - Raw Materials  
  - Pattern  
  - Work  
  - Suitable For  
- **Search Functionality** – Quickly search and filter products.
- **Collapsible Filter Menu** – Show or hide filters with animated arrow icons.
- **Responsive Grid Layout** – Built with CSS Grid & Flexbox for all screen sizes.
- **Reusable Components** – Header, Footer, Sidebar, ProductCard, etc.
- **Modern Styling** – Clean, minimalist design using custom CSS.

---
